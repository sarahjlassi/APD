Weather Forecast

Geminate comes with a feature of weather information, weather notification and alerts based on natural calamities. weather information mainly covers current precipitation, humidity, wind, sunset, sunrise and temperature of current logged in user's location. it also provides information of daily, hourly weather apart from the current climate.

It will automatically notify you for natural calamities like floods, hurricanes, tornadoes, volcanic eruptions, earthquakes, tsunamis, storms, and other geologic processes by email and notification icon on the top menu bar.

Now you can always find information about any upcoming natural calamities on your screen instead of looking up on any 3rd party weather websites. it will automatically send you upcoming high alerts or warnings on your mailbox.







Editions Supported

1. Community Version

2. Enterprise Version

Note: This module is developed, designed and tested on odoo vanila.

Available Key Features

  • Upcoming natural calamities alerts and notification.
  • Get instant information about the weather.
  • Get current information about rainfall, humidity, wind, sunset, sunrise and temperature of current logged in user's location.
  • Gives you the weather information in two parts. 1). Hourly Forecast (Upto 2 days) 2). Daily Forecast (Upto 8 days).
  • Send automated mail to a user for upcoming natural calamities.
  • Easy to mute alerts whenever you want.
  • Multi language support

Screenshots :

1. This app works on the openweathermap api. so first you need to get api from openweathermap.org otherwise you will get the following warning message.
2. Set weather API Key under general settings.

Goto Setting -> General Settings -> Weather Forecast.

3. Before enabling upcoming natural calamities alerts and notification, you need to set current logged in user's contact > GeoLocation as per below screen.
4. upcoming natural calamities alerts and notification time zone depends on the user's time zone, you need to set the time zone as per the below screen.

Goto Setting -> Users & Companies -> Users ->Preferences -> Timezone.

5. To know upcoming natural calamities alerts and notification, you need to check the 'Weather Alert' checkbox on a user profile.

Goto setting -> Users & Companies -> Users.

6. Now click on the weather icon and you will get the weather information.
7. Weather divided into 2 parts,        1). Hourly Forecast (Upto 2 days)        2). Daily Forecast (Upto 8days).
8. Whenever there is any data of upcoming natural calamities alerts and notification then the weather icon will start blinking automatically so active users can instantly focus on it.
9. The message appears after clicking on the blinking weather icon.
10. After getting upcoming natural calamities alerts and notifications, it will send an alert notification on a currently logged in user's email.
11. To turn off or mute the blinking weather icon and mail notification, the user needs to go to preferences and check on the 'Mute Weather Alert' checkbox.
12. It support multi langauge.
